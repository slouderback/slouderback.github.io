Media consumption data
After formatting the data gained from the form we can see that there is a strong drop off of usage of newspapers and books as the generations go on. They get replaced by digitial media in the form of websites and apps.
Media creation data
The second part of the data is showing us that no matter the generation there seems to be a shared interest in website content. However as expected older generations are the ones interested in creating content for print media. The main take away from this is that it seems website content is the most general platform for all people.
Media trust data
Average trust value:
There seems to be an accelerated drop of level as you progress to newer mediums which could be a result of the longer a platform is around the more regulations there are for it thus increasing the quality. As well as there being more time for trust to build up in the community that uses it. Right now anyone can post anything to the web or an app but to get something in the newspaper or a book it has to go through many more hoops and peer reviews.
