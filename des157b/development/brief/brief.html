Project Brief

This project will explore how different generations use media today and how they choose to create new media themselves. The project will be designed for both web and mobile thus requiring it to be responsive to all sizes.

Project goal

The final product will be a visually interesting display of data that makes obvious some relationship between consumption and creation.

Methods

I will use chart.js to create the charts/plots such as bar graphs, pie charts, and doughnut charts. For color I plan on allowing different color schemes as a user choice however the format will be a two color scheme. One color will be a primary and the other will be the accent.

UX

User interaction will include the color selection and hover over charts to display more information given the context.

UX goal

The user experience will be a scollable page that guides them through a data discovery process.
