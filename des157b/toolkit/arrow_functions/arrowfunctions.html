Greets User

Function takes in a name and prints out a greeting to the console.

Named Function

            function greeting(name){
                console.log("Hi", name);
            }

Function Expression

            const printGreeting = function(name){
                console.log("Hi", name);
            }

Arrow Function

            const printGreeting = (name) => {
                console.log("Hi", name);
            }
